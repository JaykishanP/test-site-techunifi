    • Each rack should list each device.
    • Draw out the room and dictate where the rack should be placed in the room and where it will face.

    Schedules
    • For each cable, there should be a schedule for data drops and speaker zones.
    • Include the destination and source for each cable.
    • Include the type and color of the cable.
    • For phone systems, these will go on a 66 block; find a way to show these will be cross-connected.
